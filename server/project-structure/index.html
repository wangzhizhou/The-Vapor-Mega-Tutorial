    VaporдҪҝз”ЁSwift Package Manager(SPM)пјҢе®ғжҳҜдёҖдёӘдҫқиө–з®ЎзҗҶзі»з»ҹпјҢзұ»дјјдәҺiOSе№іеҸ°дёҠзҡ„CocoapodsпјҢз”ЁжқҘй…ҚзҪ®е’Ңжһ„е»әvaporеә”з”ЁгҖӮдҪҝз”ЁSPMз®ЎзҗҶзҡ„йЎ№зӣ®еңЁ.gitignoreж–Үд»¶дёӯеҝҪз•Ҙе’ҢXcodeзӣёе…ізҡ„ж–Үд»¶пјҢеҰӮжһңжғідҪҝз”ЁXcodeпјҢжҜҸж¬Ўжӣҙж”№йғҪйңҖиҰҒйҮҚж–°з”ҹжҲҗйЎ№зӣ®ж–Үд»¶гҖӮ

    дёҖдёӘSPMйЎ№зӣ®е®ҡд№үеңЁPackage.swiftжЎҶжһ¶ж–Үд»¶дёӯпјҢиҝҷдёӘж–Үд»¶дёӯеЈ°жҳҺдәҶзӣ®ж Ү(Target)гҖҒдҫқиө–(Dependencies)д»ҘеҸҠжҖҺж ·жҠҠе®ғд»¬й“ҫжҺҘеңЁдёҖиө·гҖӮйЎ№зӣ®зҡ„еёғеұҖд№ҹе’Ңдј з»ҹзҡ„XCodeйЎ№зӣ®жңүжүҖдёҚеҗҢгҖӮ

    Sourceзӣ®еҪ•з”ЁжқҘеӯҳж”ҫжәҗж–Үд»¶пјҢжҜҸдёҖдёӘеңЁPackage.swiftдёӯе®ҡд№үзҡ„жЁЎеқ—йғҪеҜ№еә”Source зӣ®еҪ•дёӢзҡ„дёҖдёӘжЁЎеқ—еӯҗзӣ®еҪ•гҖӮжЁЎжқҝйЎ№зӣ®дёӯеЈ°жҳҺдәҶдёӨдёӘжЁЎеқ—: Appе’ҢRunгҖӮ

    RunжЁЎеқ—дёӢд»…жңүдёҖдёӘmain.swiftж–Үд»¶пјҢиҝҷжҳҜswiftйЎ№зӣ®зҡ„еҗҜеҠЁе…ҘеҸЈгҖӮжҲ‘д»¬еңЁејҖеҸ‘иҝҮзЁӢдёӯдёҚдјҡдҝ®ж”№RunжЁЎеқ—гҖӮ

    Testsдёӯеӯҳж”ҫжүҖжңүзҡ„жөӢиҜ•жЁЎеқ—пјҢдёҖдёӘжөӢиҜ•жЁЎеқ—еҜ№еә”дёҖдёӘжЁЎеқ—еӯҗзӣ®еҪ•гҖӮ
